## 协议支持详解

### 支持 Reality 协议的客户端

**桌面端:**
- Clash Meta内核: ClashX.Meta、Clash Verge Rev、Clash Nyanpasu、Mihomo Party、Clash Mi、OpenClash、ShellClash、Passwall
- Xray内核: V2rayNG、V2rayN、Nekoray
- Sing-box内核: Sing-box、Hiddify、NekoBox、Pantheon

**移动端:**
- iOS: Shadowrocket、Stash、Loon、Karing、Egern
- Android: ClashMetaForAndroid、NekoBox、Hiddify、Karing

### 不支持 SSR 协议

- Surge
- Surfboard
- Sing-box系列（Hiddify、GUI.for.SingBox等）

### 支持 Hysteria2 和 Vless

**Windows:** Clash Verge Rev、Clash Nyanpasu、GUI.for.Clash、FlClash、Nekoray、V2rayN、Hiddify、Karing

**Mac:** 同Windows，另加ClashX.Meta、Surge（仅Hysteria2）

**iOS:** QuantumultX（不支持Reality）、Shadowrocket、Loon、Surge、Stash、Egern、Karing、Clash Mi

**Android:** NekoBox、ClashMetaForAndroid、Hiddify、Karing、FlClash

### 支持 AnyTLS 协议

**iOS:** Shadowrocket、ClashMi、SFI

**macOS:** ClashX.Meta、SFM、Sparkle、GUI.for.Singbox、FClash

**Android:** SFA、CMFA、HUSI、NekoBox、FClash、ClashBox

**Windows:** Sparkle、GUI.for.Singbox、FClash

**Linux:** Sparkle、GUI.for.Singbox

## 各平台推荐方案

### Windows 推荐

1. **新手**: Hiddify（最简单）或 Clash Verge Rev 1.7.7
2. **进阶**: Clash Verge Rev + 自定义规则
3. **自建**: V2rayN 或 Nekoray
4. **游戏**: Netch（已停更但仍可用）

### macOS 推荐

1. **新手**: Hiddify 或 Clash Verge Rev
2. **进阶**: ClashX.Meta 或 Clash Verge Rev
3. **专业**: Surge（功能最强但最贵）
4. **自建**: V2rayU

> M1/M2 Mac 可直接使用 iOS 版 Shadowrocket

### iOS 推荐

**付费首选:**
- **性价比**: Shadowrocket $2.99（最常用）
- **功能强**: Loon $7.99 或 QuantumultX $9.99
- **专业级**: Surge $49.99

**免费首选:**
- **最简单**: Hiddify 或 Clash Mi
- **功能全**: Karing
- **轻量级**: Streisand 或 Potatso

### Android 推荐

1. **新手**: Hiddify 或 ClashMetaForAndroid
2. **进阶**: NekoBox（支持最多协议）
3. **经典**: V2rayNG
4. **美观**: Surfboard（不支持SSR）

### Linux 推荐

1. **GUI**: Clash Verge Rev 或 Clash Nyanpasu
2. **命令行**: Shell Clash
3. **轻量**: Hiddify

## 选择建议

### 按需求选择

- **追求简单**: Hiddify、Shadowrocket、Clash Mi
- **追求功能**: QuantumultX、Loon、Surge
- **追求免费**: Karing、Clash Mi、Streisand、Potatso
- **追求稳定**: Clash 系列、V2Ray 系列
- **自建节点**: V2rayN/V2rayNG、Nekoray

### 按协议选择

- **需要 Reality**: 选择支持 Clash Meta、Xray 或 Sing-box 内核的客户端
- **需要 SSR**: 避免 Surge、Surfboard、Sing-box 系列
- **需要 Hysteria2**: 大部分现代客户端都支持

## 注意事项

1. **iOS 应用购买**: 需要外区 Apple ID，可通过第三方平台（如果果商店）购买
2. **协议兼容性**: 购买机场前务必确认客户端支持的协议
3. **更新频率**: 优先选择活跃维护的开源项目
4. **隐私安全**: 优先选择开源工具，避免闭源软件
5. **版本选择**: 某些软件的特定版本更稳定（如 Clash Verge Rev 1.7.7）

## 下载资源

- **GitHub Release**: 大部分开源软件的官方下载渠道
- **App Store**: iOS/Mac 应用商店（需外区账号）
- **官方网站**: 部分商业软件的官方下载页面
- **第三方平台**: 果果商店等提供 iOS 应用租赁/购买服务
